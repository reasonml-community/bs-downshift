## Install and setup

#### yarn

```bash
$ yarn add bs-downshift
```

#### bsconfig

Add `bs-downshift` to your `bs-dependencies`: **bsconfig.json**

```json
"bs-dependencies": [
  "bs-downshift",
  "reason-react"
]
```
